# Simple user API webserver

It runs a http server with user API.

## Functionality

1. Run http-server
2. Create a user
3. Get a user

## Installing

```
git clone TODO
cd ece-userapi
npm install
```

## Usage 

1. To run http-server

```
npm start
```

2. To run tests

```
npm test
```
